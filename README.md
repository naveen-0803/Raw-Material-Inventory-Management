# Raw-Material-Inventory-Management

#A lightweight Raw Material Inventory Management web app built using HTML, CSS, JavaScript and JsonPowerDB(JPDB).

#Overview This project is a Raw Material Inventory Management System that performs basic CRUD (Create, Read, Update, Delete) operations using JsonPowerDB as the backend database.

it allows you to: Add new Raw Material records Retrieve and view Raw Material data Edit existing records Navigate between records (First, Previous, Next, Last) Store data in real-time using JPDB’s powerful REST API

All operations are done without writing any server-side code--everything runs directly in the browser using the JPDB API.

#Key Features

1.Dynamic i.Add, edit or navigate Raw Material records easily ii.Auto-detects if the database is empty on startup iii.Automatically loads the last record on the page 2.JsonPowerDB Integration i.NoSQL databse ii.Real-time performance iii.Schema-free, easy to use and developer-friendly 3.LocatStorage Usage i.Remember record positions (First, Last, Current) ii.Enables smooth navigation between records 4.Interactive UI i.All controls enable/disabled dynamically ii.Clear user flow and validation for each field

#About JsonPowerDB JsonPowerDB is a Real-time, High Performance, Lightweight, and Simple-to-use JSON database. It provides a NoSQL database with REST API support, perfect for small projects and quick prototyping.

#Conclusion

This project demonstrate how to: Integrate JsonPowerDB in a pure front-end application Manage Raw Material records dynamically Handle form validation, navigation and data storage all in one browser-based solution
